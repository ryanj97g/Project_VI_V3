## ðŸ’» Getting Started

### Prerequisites
- Windows 10/11 (Other OS: see VI3_QUICKSTART.md)
- Rust toolchain: https://rustup.rs/
- 4GB+ RAM recommended
- Optional: NVIDIA GPU

### Installation

1. **Clone this repository**
   ```bash
   git clone <repository-url>
   cd VIV3
   ```

2. **Build the project** (Windows)
   ```cmd
   build_vi3.bat
   ```
   
   Or manually:
   ```bash
   cargo build --release
   ```

3. **Run VI3** (Windows)
   ```cmd
   run_vi3.bat
   ```
   
   Or manually:
   ```bash
   cargo run --release
   ```

### Quick Commands (Windows)

| Batch File | Purpose |
